# 📖 单词复习系统

艾宾浩斯记忆曲线单词复习工具 — 翻卡 / 拼写 / 选择题

所有人共享词本，个人进度独立保存。

---

## 🚀 快速使用

### 方案一：直接访问网页（推荐）

访问部署好的网址，在浏览器中直接使用：

```
https://dr-source.github.io/word-review/
```

**iPad / 安卓平板：** 在 Safari 或 Chrome 中打开 → 添加到主屏幕 → 像真 App 一样使用

### 方案二：Windows 本地运行（无需安装）

解压后双击 `start.bat`，浏览器自动打开。

> 需要 Windows 7+，利用 PowerShell + .NET 启动本地服务，无需安装任何软件。

### 方案三：开发者模式

```bash
npm install
npm run dev      # 开发模式（热更新）
npm run build    # 构建
npm run serve    # 本地预览构建结果
```

## 📖 使用指南

### 首次使用

1. 打开网页后，首页显示欢迎界面和学习统计
2. 点击"开始背诵"进入学习
3. 支持三种模式：**翻卡** / **选择题** / **拼写**

### 翻卡背诵

1. 卡片正面显示单词 → 点击或按 **空格** 显示答案
2. 看到释义后，三选一评价：
   - **1 - 不认识**：5 分钟后重试
   - **2 - 模糊**：降低等级，30 分钟后复习
   - **3 - 认识**：提升等级，按艾宾浩斯曲线安排下次复习
3. 支持 **左右滑动**（手机）或 **键盘快捷键**（桌面）

### 选择题模式

看单词，从 4 个选项中选出正确的释义。

### 拼写模式

看释义，打字输入对应的英文单词，自动判对错并发音。

### 统计数据

首页和统计页展示：
- 总单词数 / 已掌握 / 掌握率
- 今日学习数 / 连续学习天数
- 单词掌握度分布图（Lv.0 ~ Lv.7）

## 🔐 管理员模式

> 单词管理功能默认对普通用户隐藏，只有管理员能操作。

**进入管理员模式：** 快速点击顶部标题 **"📖 单词复习"** 5 次

- 侧栏会出现"单词管理"菜单
- 底部出现"管理员"标签
- 可添加/删除词本、增删单词、批量导入、导出数据

**退出管理员模式：** 点击侧栏底部的"管理员"标签

> 管理员状态保存在当前浏览器的 localStorage 中，仅自己可见。

---

## ☁️ 数据说明

| 数据类型 | 存储位置 | 说明 |
|---------|---------|------|
| 词本 / 单词 | Supabase 云端 | 所有人共享，增删即时同步 |
| 掌握等级 / 复习时间 | 浏览器 localStorage | 个人独立，不同设备不同进度 |
| 错题本 | 浏览器 localStorage | 个人独立，仅自己可见 |
| 学习记录 / 连续天数 | 浏览器 localStorage | 个人独立，仅自己可见 |

### 数据备份

单词管理（管理员模式）→ 数据管理 → **导出为 JSON**，生成的文件包含所有词本和单词。

发给别人 → 对方在相同页面点 **从 JSON 恢复**。
